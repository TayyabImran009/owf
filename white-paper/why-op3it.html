The Optimize component  has the same features as Microsoft’s Project, Oracle’s Primavera, Atlassian’s Jira, and other popular project management scheduling tools except for the following unique features:

     

  1. Full Activity Support The Optimize component supports all organizational activities. These activities include all work accomplished by an organization. The Optimize component divides the activities into processes, products, and projects. At the same time, all other tools support only projects. The Optimize component integrates requirements with scheduled tasks for processes, products, and projects. In effect, The Optimize component is measuring and prioritizing all organizational activities daily.

    2.  

  2.   Requirements The Optimize Component

    • Supports the requirements specification for each distinct process, product, project, and issue. Some tools, such as Requisite Pro and DOORS, support requirements but not scheduling. Tools like Microsoft’s Project and Oracle’s Primavera do not support requirements. There are tools such as Requisite Pro and DOORS which do support requirements but not scheduling. The Optimize component integrates requirements with scheduled tasks for processes, products, projects, and issues.
    • Analyzes requirements assuring stakeholders’ common understanding.
    • Verifies the testing of each low-level requirement.
    • Verifies the implementation of each low-level requirement.

     

  3. Measurement The Optimize component measures staff members’ performance for each labor category they support. It computes the average hourly salary for each labor category and cost-effectiveness using each user’s measured performance. It does this using estimated effort for tasks versus actual hours to complete the effort while integrating the performance’s quality.

    4.  

  4. Ranking The Optimize component ranks each process, product, project, and issue using 32 built-in criteria plus organization-defined criteria to continuously determine how important each process, product, project, and issue’s task are relative to other tasks.

    5.  

  5. Resource Allocation Daily, the Optimize component allocates an organization’s critical tasks to its most productive resources using its patented resource allocation algorithm.

    6.  

  6. Analysis The Optimize component analyzes requirements and schedules and warns management as soon as possible when tasks are behind schedule, or costs exceed their estimates so management may take timely mitigation actions.

    7.  

  7. Performance Analysis The Optimize component analyzes each staff’s performance based on scientific data, removing biases or intuitive judgments relative to evaluating staff members.

    8.  

  8. Technical Gap Fulfillment The Optimize component analyzes each staff’s performance based on scientific data, removing biases or intuitive judgments relative to evaluating staff members.
